Lesson 15: Tableau Intro
========================

Learning Objectives
-------------------

One of the primary business intelligence tools in today’s market is Tableau. 
Using Tableau Public, we’ll introduce the basics for data visualization analyses.

* Understand the various Tableau environments: Public, Desktop, Online, and Server 
* Explain Tableau’s layout and workbook functions 
* Know how to connect to data and what data files are accepted with Tableau Public 
* Know the difference between a Dimension and Measure and how Tableau automatically interprets the best visualization
* Understand data types and how/when to change data types 
* Join and blend data sources with Tableau and Tableau Interpreter 
* Create basic visualizations with bar charts, line graphs, and others
