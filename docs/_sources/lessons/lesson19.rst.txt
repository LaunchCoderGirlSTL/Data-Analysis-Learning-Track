Lesson 19: Maps
===============

Learning Objectives
-------------------

Maps create spatial visualizations that can facilitate quick data interpretation for users.

* Understand different types of maps: proportional symbol maps, choropleth, point distribution, flow maps, and spider maps
* Build the various types of maps
* Design unique custom maps using spatial files like Shape, MapInfo, KML, GeoJSON and static images
